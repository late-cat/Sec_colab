# Customer Churn Prediction & Model Optimization

Welcome to the Customer Churn Prediction project! This project aims to predict whether a customer is likely to churn (leave) or stay with a service using machine learning. It features a complete pipeline from exploratory data analysis to advanced model optimization and a deployed interactive web application.

## 🚀 Advanced Model Optimization (Branch: `Model-Optimization`)
To maximize our model's reliability and performance, we've implemented an advanced optimization pipeline in `notebooks/01_model_optimization.ipynb`. This notebook includes:
- **Beginner-Friendly Format**: Small, highly readable cells executing step-by-step.
- **SMOTE (Synthetic Minority Over-sampling Technique)**: To effectively handle class imbalance between churned and retained customers.
- **Hyperparameter Tuning**: Using `RandomizedSearchCV` for faster, efficient parameter tuning.
- **Stratified K-Fold Cross-Validation**: Ensuring the model's stability across different data subsets.
- **Deep Error Analysis**: Detailed comparative tables, Confusion Matrices, ROC-AUC, and Precision-Recall curves.
- **Model Persistence**: Automatically saving the best performing model and scaler directly to the `models/optimized-models/` directory for deployment.
